# Altin's T-Shirt Webshop


Dit is een studieproject voor **KDG Hogeschool**, waarin ik een T-shirt webshop heb gemaakt om mijn webontwikkelingsvaardigheden te tonen. Het project is gebouwd met **HTML, CSS en JavaScript**.


## Functies


- Productoverzicht en detailpagina’s
- Interactieve winkelwagen (items toevoegen/verwijderen)
- Visueel aantrekkelijke layout met CSS
- Shop waar je kan zoeken en filteren


## Installatie


1. Clone de repository:
```bash
git clone https://github.com/Altin-Recica/WebshopProjectKDG.git
```
2. Open `index.html` in je browser.


## Technologieën


- HTML
- CSS
- JavaScript
